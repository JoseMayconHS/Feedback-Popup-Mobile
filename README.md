
# NLW Return Mobile

Um projeto de um popup para enviar feedback do usuário.


## Tecnologias

- Typescript;
- React Native;
- Styled-Components
## Funcionalidades

- Screenshot automático da tela;


## Stack utilizada

**Front-end:** React, TailwindCSS

**Back-end:** Node, Express

**Mobile:** React Native, Styled-Components
